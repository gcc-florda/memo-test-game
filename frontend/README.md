# MemoTest Game

## About The Project

Simple memo test game made for kids.

## Installetion

1. Clone the repo
```sh
git clone https://github.com/gcc-florda/memo-test-game.git
```

2. Enter the repository folder
```sh
cd memo-test-game
```

3. Install npm packages
```sh
npm i
```

4. Run the project
```sh
npm start
```
